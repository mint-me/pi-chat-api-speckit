# FAQ

## Is `422 Unprocessable Entity` correct in FastAPI?

Yes, for request validation failures (missing/invalid body/query/path fields),
FastAPI returns `422` by default. `/docs` itself should return `200`.

## Why does `alembic/script.py.mako` exist?

Alembic uses `script.py.mako` as the template when generating new migration
files. It is expected in Alembic-based projects.

## Is it okay to have `__init__.py` files everywhere?

Yes. They mark Python packages explicitly and keep imports predictable.

## What about `__pycache__` directories?

They are generated bytecode caches at runtime and should not be committed. This
repo ignores them via `.gitignore`.

## How can I see mock vs real LLM output in smoke tests?

Run:

```bash
make smoke SMOKE_ARGS=--show-stream
```

If `OPENROUTER_API_KEY` is set, stream comes from OpenRouter; otherwise from
the deterministic mock provider.

## Why is TDD commit history not strict in one commit pair per test/code step?

This repository now enforces the workflow expectation in docs: write/adjust
tests before behavior changes when feasible, and include test evidence in each
meaningful commit.
